# Baiao Trading

## 项目简介

这是一个面向公众展示的网页行情分析平台项目，服务于四川百傲科技有限公司的市场分析产品展示与技术能力说明。

项目当前围绕网页端行情分析场景进行开发，目标是将实时市场数据、技术指标、交互图表与结构化分析方法整合到统一界面中，对外提供高级图表展示与技术支持能力。

## 这个项目用来做什么

本项目主要用于构建一个公开可访问的行情分析网站，帮助用户在网页中完成以下工作：

- 查看多市场行情数据与图表信息
- 使用交互式图表进行价格结构观察
- 结合技术指标辅助理解市场走势
- 借助缠论相关方法，对走势结构、中枢、趋势延续与转折进行教学性分析
- 对外展示产品能力、技术路线与高级图表集成方案

它不是交易执行系统，不提供下单、撮合、账户托管或资金管理功能，重点在于行情展示、技术分析和研究支持。

## 核心方向

- 实时或准实时市场数据接入
- 技术指标与结构化行情解读
- 高级交互图表展示
- 基于缠论的行情分析表达
- 面向公开访问的产品介绍与技术支持页面

## 当前项目结构

- `index.html`：首页
- `market-analysis.html`：行情分析主页面
- `about.html`：关于页面
- `contact.html`：联系页面
- `terms.html`：使用条款页面
- `zh/`：中文页面目录
- `assets/`：样式、脚本、图片等静态资源
- `api/`：市场数据接口说明文件
- `server.js`：本地静态服务与市场 API 代理入口
- `vendor/`：本地高级图表相关预留目录

## 技术说明

当前项目采用静态网页加本地 Node.js 服务的方式组织：

- 前端使用 HTML、CSS、JavaScript 构建公开网站
- 本地服务使用 Node.js 启动
- 图表层预留高级图表集成能力
- 市场数据侧使用本地 API 代理与外部数据源对接

## 本地运行

安装依赖后可直接启动本地服务：

```bash
npm install
npm start
```

默认入口为本地网站首页，服务脚本定义在 `package.json` 中。
